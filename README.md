# Personal Website

This is a static GitHub Pages personal homepage based on the academic homepage style used by Siqi Shang and adapted from Jon Barron's template.

## Edit Your Content

- Replace the placeholders in `index.html`.
- Replace `images/profile.svg` with your own profile image, or update the image path in `index.html`.
- Add publication/project images under `images/` and use `<img class="thumb" src="images/your-image.png" alt="...">`.
- Use `class="highlight"` on publication rows you want to emphasize.

## Publish on GitHub Pages

1. Create a repository named `YOUR_GITHUB_USERNAME.github.io`.
2. Push these files to the repository's `main` branch.
3. GitHub Pages will serve the site at `https://YOUR_GITHUB_USERNAME.github.io/`.

If you use a different repository name, enable Pages in the repository settings and publish from the `main` branch root.
